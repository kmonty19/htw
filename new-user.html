New User Guide

Welcome to the HTW Data platform! This guide will help you navigate and make the most of the Hawaii Tech Week 2025 audience dataset.

Getting Started

Navigation

  • Home - Overview and statistics about the dataset
  • New User - This guide page
  • Dataset - The main data table with search and filter capabilities
Quick Tip: Click the menu icon (☰) in the top-right corner to access navigation at any time.

Using the Dataset

Searching and Filtering

  • Use the search box to filter rows by any text content
  • Search across all columns simultaneously
  • Results update in real-time as you type

Sorting Data

  • Click any column header to sort by that column
  • Click again to reverse the sort order
  • Look for ▲ (ascending) or ▼ (descending) arrows

Data Columns

  • Role / Job Title - Professional role or position
  • City - Location city
  • State / Province - State or province
  • Country - Country location
  • Industry - Industry sector or field
  • CONFIRM_TIME - Registration confirmation timestamp
  • LAST_CHANGED - Last data update timestamp

Search Tips

Effective Search Strategies

  • Industry Search: Try terms like "software", "healthcare", "education"
  • Location Search: Search for "Honolulu", "HI", "USA", or specific cities
  • Role Search: Look for "CEO", "engineer", "manager", "analyst"
  • Combined Search: Use multiple terms like "AI engineer" or "Honolulu software"
Pro Tip: The search is case-insensitive and searches across all visible text in each row.

Mobile Usage

The platform is fully responsive and works great on mobile devices:

  • Tables scroll horizontally on smaller screens
  • Touch-friendly interface elements
  • Optimized search and navigation for mobile

Need Help?

If you encounter any issues or have questions about the data:

  • Check the Home page for dataset overview and statistics
  • Try different search terms if you're not finding what you're looking for
  • Remember that the data represents HTW 2025 participants and may include diverse professional backgrounds
